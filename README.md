### 烘焙小tips

#### **餅乾**
* 奶油比例
    * 多：酥脆
    * 少：乾
* 糖
    * 比例
        * 高：脆
        * 低：乾
    * 種類
        * 砂糖：脆硬
        * 糖粉：酥鬆
* 蛋液比例
    * 純蛋黃：酥鬆
    * 全蛋：介於中間
    * 純蛋白：脆硬
